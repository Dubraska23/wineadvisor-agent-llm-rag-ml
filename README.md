# wineadvisor-agent-llm-rag-ml
Agente autónomo de IA (WineAdvisor) que integra Machine Learning, RAG (Retrieval-Augmented Generation) y Tools para el análisis enológico. Capaz de predecir calidad, justificar decisiones mediante consulta técnica vectorizada y devolver salidas estructuradas validadas. Un ejemplo de IA Agéntica con gobernanza y rigor técnico (UAM 2026).
